The Code for Flashback

                    
                        //Get the string from the page
                        //controller function
                        function getValue(){
                        
                            document.getElementById("alert").classList.add("invisible");
                            
                            let userString = document.getElementById("userString").value;
                        
                            let revString = reverseString(userString);
                        
                            displayString(revString);
                        }
                        
                        //Reverse the string
                        //logic function
                        function reverseString(userString){
                        
                            let revString = [];
                        
                            //reverse a string using a for loop
                            for (let index = userString.length - 1; index >= 0; index--) {
                        
                                revString += userString[index];        
                            }
                        
                            return revString;
                        }
                        
                        //Display the reversed string to the user
                        //view function
                        function displayString(revString){
                        
                            //write the message to the page
                            document.getElementById("msg").innerHTML = `Your string reversed is: ${revString}`;
                        
                            //show the alert box
                            document.getElementById("alert").classList.remove("invisible");
                        }

Flashback is a program that consists of three functions that reverse a string.

getValue()

The getValue function controls how the program is executed. It starts of by ensuring that the alert is invisible to the user upon execution. Once it gets past that stage it then defines a variable and gathers the users input from the page. After the user input has been collected another variable is declared and we pass the userString variable into the reverseString function. The reverseString function will give us our intended output and then the getValue function will pass that variable into the displayString function.

reverseString()

Just like the getValue() function we start off declaring a variable but this time it is an empty array. The empty array is then used in a for loop to then reverse our users input. We do this by getting the length of the string and then starting at the end. During this process the loop will begin to decrement and push the characters into our revString array. At the end of the for loop we then return revString to be used later in the program.

displayString()

This function is called by the getValue() function and uses data from the reverseString() function. Inside this function we use a template literal to display a message to the user. It then removes the "invisible" class from the alert on the page which contains our reversed string.
